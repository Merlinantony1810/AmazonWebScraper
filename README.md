# Amazon Web Scraper

This Jupyter notebook contains a web scraper that monitors the price of a product (PS5 in this case) on Amazon and sends an email notification when the price falls below ‚Çπ50,000.

## Features:
- Scrapes the price and title of a PS5 product from Amazon.
- Saves data (product title, price, and date) to a CSV file.
- Sends an email notification when the price falls below ‚Çπ50,000.

## Requirements:
- Python 3.x
- Libraries: `beautifulsoup4`, `requests`, `pandas`, `smtplib`
